📊 البيانات المطلوبة
════════════════════════════════════════════════════════════════════════

عند إضافة مستخدم تحتاج:

  🔑 معرف المستخدم (User ID)
     └─ رقم من 6-10 أرقام
     └─ معرفك في Telegram
     └─ مثال: 123456789

  👤 اسم المستخدم (Username)
     └─ بدون علامات خاصة
     └─ مثال: ahmad_user

  📛 الاسم الأول (First Name) - اختياري
     └─ مثال: أحمد


════════════════════════════════════════════════════════════════════════
🎛️ خيارات لوحة التحكم
════════════════════════════════════════════════════════════════════════

في ADMIN_PANEL.bat ستجد:

  1️⃣ إضافة مستخدم جديد
     └─ يطلب: User ID, Username, First Name
     └─ النتيجة: مستخدم جديد مع Trial 3 أيام

  2️⃣ تفعيل الأدمن
     └─ يطلب: User ID
     └─ النتيجة: المستخدم أصبح أدمن

  3️⃣ إزالة الأدمن
     └─ يطلب: User ID
     └─ النتيجة: إزالة صلاحيات الأدمن

  4️⃣ عرض قائمة الأدمن
     └─ يعرض: جميع الأدمن الحاليين

  5️⃣ ترقية مستخدم
     └─ يطلب: User ID, اختيار الباقة
     └─ الباقات: bronze/silver/gold/platinum

  6️⃣ عرض بيانات مستخدم
     └─ يطلب: User ID
     └─ يعرض: كل معلومات المستخدم

  7️⃣ عرض جميع المستخدمين
     └─ جدول بجميع المستخدمين

  8️⃣ حذف مستخدم
     └─ يطلب: User ID + تأكيد

  9️⃣ تصدير تقرير
     └─ ينشئ ملف Excel بكل البيانات

  0️⃣ خروج

📝 أمثلة عملية
════════════════════════════════════════════════════════════════════════

مثال 1: إضافة موظف جديد وتفعيله أدمن
─────────────────────────────────────
  1. افتح ADD_USER.bat
  2. أدخل:
     User ID: 987654321
     Username: mohammad_team
     First Name: محمد
  3. اختر Yes للأدمن
  ✅ النتيجة: موظف جديد بصلاحيات أدمن!


مثال 2: ترقية عميل من Trial إلى Gold
────────────────────────────────────
  1. افتح ADMIN_PANEL.bat
  2. اختر [5] ترقية مستخدم
  3. أدخل المعرف: 123456789
  4. اختر: gold
  ✅ النتيجة: عميل صار عنده 365 يوم!


مثال 3: عرض بيانات مستخدم
──────────────────────────
  1. افتح ADMIN_PANEL.bat
  2. اختر [6] عرض بيانات
  3. أدخل المعرف: 123456789
  ✅ النتيجة: كل البيانات واضحة!
